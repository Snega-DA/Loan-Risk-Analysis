# Loan-Risk-Analysis
## Overview

This project focuses on analyzing loan data to understand borrower behavior and identify key factors influencing loan defaults. Using Python and pandas, the dataset is loaded, cleaned, and explored to generate meaningful financial insights. The final output is a structured analysis report that highlights trends, patterns, and risk indicators.

## Dataset

The dataset contains information related to loan applicants and loan performance, including Loan status,Interest rate,Debt-to-Income ratio (DTI),Home ownership,Loan amount and purpose

## Tools & Technologies

Python                               
Pandas (data manipulation & reporting)                                 
NumPy (numerical operations)                         
Matplotlib / Seaborn (data visualization)                                 

## Project Steps

Data Loading : Imported dataset and performed initial checks                                                                               
Data Cleaning : Handled missing values and ensured correct data types                                                              
EDA : Explored distributions, trends, and relationships                                                                                          
Analysis : Calculated key metrics and compared borrower segments                                                                                     
Reporting : Summarized findings using tables and visualizations                                                                                         

## Dashboard

Basic visualizations were created using Python to represent:                                           
Loan status distribution                                                                     
Risk patterns across borrower categories                                                                             
Financial indicators such as DTI and interest rate                                             

## Results
Identified factors contributing to higher loan default risk
Observed relationships between DTI, interest rate, and loan status
Highlighted borrower segments with higher financial risk
Provided data-driven insights for better decision-making

## Conclusion

This project demonstrates how data analysis techniques can be applied to financial datasets to uncover insights, assess risk, and support informed decision-making.
